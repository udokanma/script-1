ABOUT HTML

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displaced in a web browser.

Types

  1. Transitional.
  2. Strict.
  3. Frameset.

Uses

  • Game development.
  • Responsive designs.
  • Creation of web documents.
  • offline web application.
  • Data entry support.many more....

Advantages

  • it is easy to learn.
  • Every browser supports HTML Language.
  • HTML is light weighted and fast to load.
  • Storage of big files are allowed because of the application cache feature.
